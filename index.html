Политика конфиденциальности

1. Сбор данных
NEVIO не собирает и не хранит ваши личные данные на серверах. Вся история хранится только на вашем устройстве.

2. Запросы к ИИ
Ваши запросы передаются в ИИ через защищённое соединение для генерации ответов. Содержимое диалогов не сохраняется на наших серверах.

3. Telegram
Используется только имя и фото профиля для персонализации. Никакие данные не передаются третьим лицам.

4. Ваши права
Вы можете в любой момент очистить историю или сменить роль через настройки.

Условия использования

1. Принятие условий
Используя NEVIO, вы соглашаетесь с данными условиями.

2. Ответственность
NEVIO предоставляет ответы на основе ИИ. Всегда проверяйте критически важную информацию самостоятельно.

3. Ограничения
Запрещается использовать приложение для незаконных действий, распространения вредоносного контента.

4. Изменения
Условия могут обновляться. Продолжение использования означает принятие новых условий.
