Политика конфиденциальности

1. Сбор данных
NEVIO не собирает и не хранит ваши личные данные на серверах. Вся история хранится только на вашем устройстве.

2. Обработка запросов
Ваши запросы передаются анонимно в искусственный интеллект для генерации ответов. Мы не сохраняем содержимое ваших диалогов на своих серверах.

3. Telegram
Приложение интегрируется с Telegram только для отображения вашего профиля (имя, фото). Никакие данные не передаются разработчику.

4. Ваши права
Вы можете в любой момент очистить историю или удалить аккаунт через настройки. Все данные будут безвозвратно удалены с вашего устройства.

5. Контакты
По вопросам конфиденциальности вы можете обратиться через поддержку Telegram-бота.

Используя NEVIO, вы соглашаетесь с данными условиями.

2. Ответственность
NEVIO предоставляет информацию на основе искусственного интеллекта. Мы не несём ответственности за возможные ошибки в ответах. Всегда проверяйте критически важную информацию.

3. Ограничение использования
Запрещается использовать приложение для незаконных действий, спама, оскорблений.

4. Изменения
Мы можем обновлять условия в любое время. Продолжение использования означает принятие новых условий.
