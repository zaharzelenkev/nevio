Политика конфиденциальности

1. Общие положения

Настоящая Политика конфиденциальности определяет порядок обработки и защиты информации о пользователях приложения Nevio (далее — «Приложение»).

2. Какие данные мы собираем

Приложение не собирает, не хранит и не передает третьим лицам персональные данные пользователей. Вся информация (имя, роль, школа, класс) сохраняется исключительно на устройстве пользователя в локальном хранилище браузера и может быть удалена пользователем в любой момент.

3. Обработка сообщений

Все запросы, отправляемые в Приложение, передаются анонимно в API для обработки и не сохраняются на серверах Приложения. Мы не имеем доступа к истории ваших диалогов.

4. Сторонние сервисы

Приложение использует API Groq для обработки запросов. Обработка данных осуществляется в соответствии с политикой конфиденциальности Groq.

5. Ваши права

Вы можете в любой момент удалить все свои данные, выйдя из аккаунта или очистив данные приложения в настройках Telegram.

6. Контакты

По вопросам, связанным с конфиденциальностью, вы можете связаться с нами через Telegram-бота.

7. Изменения

Мы оставляем за собой право обновлять данную политику. Актуальная версия всегда доступна в настройках Приложения.
